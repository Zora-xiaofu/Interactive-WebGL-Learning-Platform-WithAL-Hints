# WebGL Learning Platform with AI-Assisted Hints

An interactive educational platform designed to help beginners learn the WebGL rendering pipeline through step-by-step tutorials, hands-on coding exercises, and AI-assisted contextual hints.

This project was developed as my Bachelor's Final Year Project at the University of Manchester. It combines educational website design, full-stack web development, user experience (UX) principles, and OpenAI-powered learning assistance to improve beginners' learning experience.

## Features

- Step-by-step rendering pipeline tutorials
- Interactive coding playground
- GPT-powered contextual hints
- Real-time rendering output and error console
- Progressive disclosure learning design
- User authentication (Sign up & Login)
- SUS-based usability evaluation
